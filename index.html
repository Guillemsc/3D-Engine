SuSto Engine

by Guillem Sunyer and Simón Stoyanov

Description

3D Engine made by Guillem Sunyer Caldú and Simón Stoyanov Beltrán for the 3D engines subject during the 3rd year of Game Design and Game Development degree at CITM.

The engine was born for the 3d engines assignment and has a User Interface subsytem.

Project

The game engine's layout is based in Unity's design and has a docking functionality.

SuSto Engine's game objects work with components, so if the user wants to use a mesh, a Mesh Component will be needed.
